# Structure du Frontend

## Fichiers à créer

1. **index.html** - Page portfolio (page publique principale)
2. **app.html** - Application annonces (liste des annonces)
3. **login.html** - Connexion
4. **register.html** - Inscription
5. **detail.html** - Détail d'une annonce
6. **create.html** - Créer une annonce
7. **edit.html** - Modifier une annonce
8. **my-annonces.html** - Mes annonces

## Dépendances

- Bootstrap 5 (CDN)
- Font Awesome 6 (CDN)
- API Client (js/api.js)
- Utils (js/utils.js)

## Notes

Le portfolio (index.html) est une page statique qui n'a pas besoin de l'API.
Les autres pages nécessitent l'API pour fonctionner.
